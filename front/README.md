## r-ct 前端

使用react.js, mongodb 和 koa 的一个简易的聊天室应用

## Install
1. cloen项目 git clone https://github.com/Joursion/r-ct.git
2. 进入目录 cd r-ct/front 
3. 安装依赖 npm install
4. 相关配置在 src/default.js
4. 运行测试 npm start
