## r-ch 后端

- 使用koa, mongodb

## Install
1. cloen项目 git clone https://github.com/Joursion/r-ct.git
2. 进入目录 cd r-ct/server
3. 安装依赖 npm install
4. 相关配置在 config.js
4. 运行测试 npm start
